Summary: Pyber Rideshare Data


The purpose of this project was to analyze all of Pyber's historic rideshare data and identify any trends.

The data set contains information about every driver, and every historic ride, including details like city, driver count, individual fares, and city type.

The first objective is build a bubble plot that showcases the relationship between 4 key variables:

  • Average Fare ($) Per City
  • Total Number of Rides Per City
  • Total Number of Drivers Per City
  • City Type (Urban, Suburban, Rural

The second objective is to build 3 pie charts that demonstrate the following:

  • % of Total Fares by City Type
  • % of Total Rides by City Type
  • % of Total Drivers by City Type
